## 🚀 编译方法

### 方法1：使用GUI工具（推荐）
1. 运行 `theos_online_builder.py`
2. 选择"编译项目"选项卡
3. 选择 `MapReplacer-Standalone` 项目
4. 点击"一键上传编译"
5. 等待GitHub Actions编译完成
6. 下载编译产物（.deb包）

### 方法2：本地编译
```bash
cd projects/MapReplacer-Standalone
export THEOS=~/theos
make package
```

### 方法3：使用构建脚本
```bash
cd projects/MapReplacer-Standalone
./build.sh
```

## 📦 安装方法

编译完成后会生成 `.deb` 包，安装方式：

1. **Filza安装**（推荐）
   - 将.deb传到设备
   - 用Filza打开并安装

2. **SSH安装**
   ```bash
   scp MapReplacer.deb root@设备IP:/var/root/
   ssh root@设备IP
   dpkg -i /var/root/MapReplacer.deb
   ```

3. **Cydia/Sileo本地源**
   - 添加本地源
   - 搜索安装

## 📱 使用方法

1. 安装完成后在主屏幕找到"MapReplacer"应用
2. 打开应用，看到欢迎界面
3. 屏幕右侧出现蓝色悬浮按钮
4. 点击悬浮按钮打开地图管理面板
5. 选择想要的地图，点击"下载"
6. 等待下载完成，自动应用
7. 可以随时恢复原始地图
